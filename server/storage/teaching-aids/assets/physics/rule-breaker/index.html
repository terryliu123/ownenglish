🧠 规则破壁者

改编自经典的“威斯康星卡片分类测验 (WCST)”

这是一个关于“逻辑”与“变通”的测试。

  • 你需要把下方的牌，归类到上方的4张牌中。
  • 分类的规则可能是：颜色、形状 或 数量。
  • 系统只会告诉你“对”或“错”。你需要通过试错来猜出当前规则。
  • ⚠️ 警告： 规则会在游戏中途悄悄改变！

*持续性错误：指规则已经改变，但你仍固执地坚持旧规则的次数。这个指标越低，代表思维越灵活。

👨‍⚕️ 心理学解读：

这个测试考察的是你的执行功能 (Executive Function)。在生活中，我们经常遇到情况变了、老办法行不通的时候。

能够迅速察觉变化、抑制旧习惯、建立新策略，是大脑年轻、高效的重要标志。
