🧠 科学原理解析

神奇的数字 7±2： 心理学家乔治·米勒发现，人类短时记忆的容量大约只有 7 个单位（5到9之间）。

在这个游戏中，当你需要记住的方块超过 5-7个 时，你会发现难度陡增，感觉脑子“不够用”了。这就是你的工作记忆容量达到了极限。
