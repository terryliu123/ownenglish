刚架桥 (Rigid Frame Bridge)

梁柱刚性连接：既像梁受弯，又像柱受压，还有拱的推力
0 kN 水平推力 (Thrust)
0 kNm 节点负弯矩 (Corner Moment)
0 kNm 跨中正弯矩 (Mid-Span)
结构特点解析：
  • 刚性节点 (Rigid Joint)： 观察横梁和立柱的转角处。当横梁向下弯曲时，它像杠杆一样“强行”带动立柱向内弯曲。
  • 内力重分布： 这种刚性连接把本来完全由横梁承担的弯矩，分了一部分给立柱（节点负弯矩）。结果是：梁的跨中弯矩变小了，梁就可以做得更薄。
  • 有推力结构： 观察柱脚的箭头。立柱倾向于向外张开，因此基础必须提供向内的水平反力。
