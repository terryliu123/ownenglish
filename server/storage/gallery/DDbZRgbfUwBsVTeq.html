实验证明，红、绿、蓝三种色光无法被分解，而且把这三种色光按不同比例混合后，可以产生各种颜色的光，因此把红、绿、蓝叫做色光的三原色。

彩色电视机画面上的丰富色彩就是由三原色光混合而成。三原色光中任意两种色光等量相加，则成为三原色光中另一种色光的补色光。

  • 等量的 红光 + 绿光 = 黄光 (互补于蓝光)
  • 等量的 红光 + 蓝光 = 品红光 (也称洋红，互补于绿光)
  • 等量的 绿光 + 蓝光 = 青光 (互补于红光)
  • 三种光等量混合 = 白光

在这个交互素材中，你可以调整右侧滑块改变红、绿、蓝三种色光的比例来观察混合出的颜色变化。
