# 氛围特效系统设计文档

## 一、需求概述

### 1.1 功能描述
课堂教学场景下，通过特效增强课堂氛围，支持自动触发和手动触发，服务于课堂各环节。学生端、教师端、大屏端均展示特效。

### 1.2 设计目标
- 增强课堂趣味性，提高学生参与感
- 服务课堂环节，答题、表扬、倒计时等场景
- 支持自动触发（根据课堂事件）和手动触发

### 1.3 三端角色

| 端 | 角色 | 功能 |
|---|---|---|
| 学生端 | 特效接收者 + 手动触发 | 可手动触发特效 |
| 教师端 | 特效控制者 | 配置特效、自动/手动触发、同步展示 |
| 大屏端 | 特效展示 | 同步展示特效 |

---

## 二、触发方式

| 触发方式 | 说明 |
|---------|------|
| 自动触发 | 根据课堂事件自动触发（如答对、倒计时结束） |
| 手动触发 | 教师或学生在端上手动选择触发 |

---

## 三、预设特效库

### 3.1 特效类型

| 特效名 | 描述 | 视觉效果 |
|-------|------|---------|
| `star_fall` | 星星/粒子飘落 | 从上往下飘落的星星粒子 |
| `confetti` | 彩色纸屑/烟花 | 彩色粒子爆发后飘落 |
| `cheer` | 欢呼 | 屏幕震动 + 欢呼图标跳动 |
| `glitch` | 故障艺术 | 闪烁、色彩错位、噪点 |
| `flash` | 全屏闪白 | 屏幕闪白后恢复 |
| `shake` | 屏幕震动 | 整个屏幕左右/上下震动 |

### 3.2 特效与课堂环节映射

| 课堂环节 | 事件 | 推荐特效 | 触发方式 |
|---------|------|---------|---------|
| 答题 | 学生答对 | `cheer` | 自动 |
| 答题 | 全班正确 | `confetti` | 自动 |
| 答题 | 学生答错 | `glitch`（轻微） | 自动 |
| 倒计时 | 时间到 | `glitch` + `flash` | 自动 |
| 表扬 | 教师表扬 | `star_fall` | 手动 |
| 互动 | 开启大屏 | `flash` | 自动 |
| 结束 | 下课 | `confetti` | 手动 |

## 四、数据结构

### 4.1 WebSocket 消息

```typescript
// 特效配置
interface AtmosphereConfig {
  type: 'atmosphere_config'
  enabled: boolean              // 总开关
  effects: Record<string, EffectSetting>  // 各事件特效配置
}

interface EffectSetting {
  effect: string      // 特效名
  enabled: boolean    // 是否启用
  auto: boolean       // 是否自动触发
}

// 特效触发
interface AtmosphereTrigger {
  type: 'atmosphere_trigger'
  effect: string              // 特效名
  source: 'student' | 'teacher' | 'system'  // 触发来源
  sourceName?: string         // 触发者姓名（可选）
}
```

### 4.2 特效配置结构

```typescript
// 示例配置
{
  "enabled": true,
  "effects": {
    "student_correct": { "effect": "cheer", "enabled": true, "auto": true },
    "class_all_correct": { "effect": "confetti", "enabled": true, "auto": true },
    "countdown_end": { "effect": "glitch", "enabled": true, "auto": true },
    "teacher_praise": { "effect": "star_fall", "enabled": true, "auto": false }
  }
}
```

## 五、三端功能设计

### 5.1 学生端

#### 入口
底部工具栏，「氛围特效」开关按钮

#### 主界面

```
┌─────────────────────────────┐
│  🎆 氛围特效 [●开]          │ ← 工具栏开关
└─────────────────────────────┘

开启后展开：
┌─────────────────────────────┐
│  🎆 手动特效               │
│  ───────────────────────  │
│  [⭐星星] [🎉欢呼] [⚡警报]  │
│  ───────────────────────  │
│  [🔴故障] [💫闪光] [📳震动]  │
└─────────────────────────────┘
```

#### 规则
| 规则 | 限制 |
|------|------|
| 发送间隔 | 5 秒/次（防刷屏） |
| 特效时长 | 3-5 秒自动结束 |
| 特效叠加 | 最多 2 个特效同时 |

### 5.3 大屏端

#### 渲染规则
| 规则 | 说明 |
|------|------|
| 特效叠加 | 最多 2 个特效同时显示 |
| 层级 | 特效层叠在白板内容之上 |
| 自动结束 | 特效 3-5 秒自动消失 |
| 响应式 | 适配不同大屏分辨率 |

#### 特效效果

**星星飘落 (star_fall)**
```

  ✦       ✦
              ✦    ✦
```

```
    ╔═══════════════════╗
    ║   🎉  CHEER! 🎉   ║
    ╚═══════════════════╝
         ↕ 震动效果
```

**故障 (glitch)**
```
    R G B 色彩错位
    ░░▒▒▓▓██ 噪点

```

## 六、交互流程

### 6.1 自动触发流程

```
课堂事件发生（学生答对）
    │
    ▼
服务端判断配置
    │
    ▼
触发 AtmosphereTrigger 广播
    │
    ├─► 学生端：渲染特效
    ├─► 教师端：渲染特效
    └─► 大屏端：渲染特效
```

### 6.2 手动触发流程

```
教师/学生点击手动触发按钮
    │
    ▼
发送 AtmosphereTrigger
    │
    ▼
各端收到消息
    │
    ▼
渲染对应特效
```

## 七、技术实现

### 7.1 前端目录结构

```
client/src/
├── features/
│   └── atmosphere/
│       ├── components/
│       │   ├── AtmosphereButton.tsx     # 工具栏按钮
│       │   ├── AtmosphereControl.tsx    # 教师端控制面板
│       │   ├── AtmospherePanel.tsx      # 学生端面板
│       │   ├── effects/
│       │   │   ├── StarFall.tsx         # 星星特效
│       │   │   ├── Confetti.tsx          # 纸屑特效
│       │   │   ├── Cheer.tsx            # 欢呼特效
│       │   │   ├── Glitch.tsx           # 故障特效
│       │   │   ├── Flash.tsx            # 闪光特效
│       │   │   └── Shake.tsx            # 震动特效
│       │   └── AtmosphereLayer.tsx      # 特效渲染层
│       ├── hooks/
│       │   └── useAtmosphere.ts         # 氛围状态管理
│       └── types/
│           └── atmosphere.ts             # 类型定义
```

### 7.2 技术选型

| 模块 | 方案 | 说明 |
|------|------|------|
| 渲染层 | DOM + CSS Animation / Canvas | 轻量、兼容性好 |
| 粒子效果 | Canvas 或 CSS | 视特效复杂度选择 |
| 状态管理 | React Context | 端内共享 |
| 消息通信 | 复用现有 WebSocket | 无新增依赖 |
