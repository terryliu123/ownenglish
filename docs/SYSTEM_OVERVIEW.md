# OwnEnglish 系统说明文档

## 1. 项目概述

**OwnEnglish** 是一款面向直播英语教师的教学辅助平台，旨在帮助教师高效管理课堂互动、课后作业，同时为学生提供沉浸式的学习体验。平台支持实时课堂互动、题库管理、学生进度追踪、口语练习、以及会员订阅管理等功能。

---

## 2. 系统架构

### 2.1 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | React 18 + TypeScript + Vite |
| **样式** | Tailwind CSS |
| **状态管理** | Zustand（客户端状态）、TanStack Query（服务端状态） |
| **路由** | React Router v7 |
| **后端** | FastAPI (Python) |
| **ORM** | SQLAlchemy 2.0（异步） |

| **WebSocket** | 原生 WebSocket（实时课堂） |
| **支付** | 微信支付（Native 扫码支付） |
| **认证** | JWT（Access Token 120分钟 + Refresh Token） |

## 3. 用户角色

系统支持三种用户角色：

| 角色 | 职责描述 |
|------|----------|
| **教师 (Teacher)** | 创建班级、管理学生、制作课前准备任务组、发布学习包、进行直播课堂、查看学习报告 |
| **学生 (Student)** | 加入班级、参与直播课堂、完成学习包、进行口语练习、查看个人学习报告 |
| **管理员 (Admin)** | 用户管理、系统设置、会员套餐管理、发送系统通知、查看活动日志 |

## 4. 核心功能模块

### 4.1 认证与授权

- **注册/登录**：邮箱注册支持验证码验证，支持学生临时访客模式
- **密码管理**：支持密码重置、修改密码
- **JWT 认证**：双 Token 机制（Access Token 120分钟 + Refresh Token 自动续期）
- **游客模式**：学生可通过学号和姓名快速进入，限时使用（2小时）

### 4.2 班级管理（教师端）

- **创建班级**：生成唯一邀请码，学生通过邀请码加入
- **学生管理**：查看班级学生列表、移除学生
- **课程设置**：关联课程、设置上课时间

### 4.3 课前准备（任务组管理）

**任务组**是教师课前准备的题目集合，支持多种题型：

| 题型 | 描述 |
|------|------|
| **单选题 (single_choice)** | 标准单选，支持图片选项 |
| **多选题 (multiple_choice)** | 支持多选答案 |
| **填空题 (fill_blank)** | 支持多填空位 |
| **判断题 (true_false)** | 对/错判断 |
| **匹配题 (matching)** | 左右配对 |
| **排序题 (sorting)** | 选项排序 |
| **阅读理解 (reading)** | 长篇阅读配问题 |
| **图片理解 (image_understanding)** | 基于图片的选择题 |
| **改错题 (error_correction)** | 句子纠错选择 |
| **情境题 (scenario)** | 场景应用选择题 |
| **实验题 (experiment)** | 嵌入第三方实验页面（iframe） |

**功能特性**：
- 手动创建题目
- AI 生成题目（基于提示词自动生成）
- AI 导入（从文本/Word/Excel 智能解析）
- 任务组分享（教师间共享题库，通过链接导入）
- 拖拽排序

### 4.4 实时课堂（WebSocket）

**教师端功能**：
- 创建课堂会话
- 从任务组发布题目
- 实时查看学生答题情况
- 答案统计与可视化
- 竞技模式：
  - **单挑模式 (duel)**：两名学生 1v1 对战
  - **班级挑战 (class_challenge)**：全班参与的抢答赛

**学生端功能**：
- 加入课堂（通过班级关联）
- 实时接收题目
- 倒计时答题
- 即时反馈答题结果
- 查看排行榜

**课堂分享功能**：
- 学生可提交图片到教师端
- 教师可查看并展示学生分享的图片
- 支持图片灯箱放大查看

### 4.5 学习包（Study Pack）

课后作业系统，包含多种模块类型：

| 模块类型 | 内容描述 |
|----------|----------|
| **词汇 (vocabulary)** | 单词、释义、音标、配图 |
| **句子 (sentence)** | 例句、翻译、句型 |
| **听力 (listening)** | 听力材料 + 问题 |
| **阅读 (reading)** | 阅读理解材料 |
| **口语 (speaking)** | 录音题，学生提交音频 |

**功能特性**：
- AI 自动生成学习包
- 支持导入外部内容
- 截止时间设置
- 学生完成进度追踪

### 4.6 口语练习

- 浏览器原生录音（MediaRecorder API）
- 最长 60 秒录音
- 音频文件上传至服务器
- 支持回放和重新录制

### 4.7 免费练习区

面向非注册用户的开放内容，A1-A2 级别：

| 分类 | 内容 |
|------|------|
| 日常问候 | 打招呼、自我介绍 |
| 自我介绍 | 姓名、年龄、国籍 |
| 数字与时间 | 数字、时间表达 |
| 礼貌用语 | 请求、道歉、感谢 |
| 问路指路 | 方向表达 |
| 购物用语 | 商店购物对话 |

### 4.8 学习报告与分析

**学生端报告**：
- 学习概览（完成率、提交数、正确率）
- 薄弱点分析
- 学习建议
- 各模块类型正确率统计

**教师端分析**：
- 班级学习概览
- 学生进度列表
- 实时课堂结果统计
- 专注学生提醒（需关注的学生列表）

### 4.9 会员与支付系统

**会员套餐**：

| 套餐 | 权益 |
|------|------|
| **Free（免费）** | 基础功能，有限额度的班级、任务组、学习包 |
| **月度会员** | 解锁更多班级、学生数、AI 生成次数 |
| **年度会员** | 月度会员的全部权益，更优惠 |

**支付方式**：
- 微信支付（Native 扫码支付）
- 支付成功后自动激活会员
- 支持查看订单历史和状态

**管理员功能**：
- 配置套餐价格和额度
- 手动为用户调整会员状态
- 查看支付订单

### 4.10 通知系统

**通知类型**：
- 系统公告
- 班级公告
- 新学习包分配
- 学习包即将到期
- 直播课开始
- 作业已批改
- 新学生加入班级
- 分享的任务组被导入

**功能特性**：
- 实时通知推送
- 已读/未读状态
- 通知历史列表
- 全部忽略功能

### 4.11 活动日志

记录用户关键行为：
- 创建任务组
- 发布任务
- 分享任务
- 创建班级
- 创建学习包
- 学生加入班级

## 5. API 端点概览

### 认证
- `POST /api/v1/auth/register` - 注册
- `POST /api/v1/auth/login` - 登录
- `POST /api/v1/auth/refresh` - 刷新 Token
- `GET /api/v1/auth/me` - 当前用户信息

### 班级
- `GET/POST /api/v1/classes` - 列表/创建
- `POST /api/v1/classes/join` - 加入班级（邀请码）

### 课前准备（任务组）
- `GET/POST /api/v1/live/task-groups` - 列表/创建
- `GET/PUT /api/v1/live/task-groups/{id}` - 详情/更新
- `POST /api/v1/live/task-groups/{id}/tasks` - 添加题目
- `POST /api/v1/live/task-groups/import` - 导入题目
- `POST /api/v1/live/task-groups/ai-generate` - AI 生成

### 实时课堂
- `WS /api/v1/live/ws` - WebSocket 连接
- `POST /api/v1/live/sessions` - 创建课堂会话
- `POST /api/v1/live/sessions/{id}/end` - 结束课堂
- `POST /api/v1/live/challenges` - 创建竞技模式

### 学习包
- `GET/POST /api/v1/study-packs` - 列表/创建
- `PATCH /api/v1/study-packs/{id}` - 更新
- `POST /api/v1/study-packs/{id}/publish` - 发布
- `POST /api/v1/study-packs/submissions` - 提交答案

### 会员
- `GET /api/v1/membership/plans` - 套餐列表
- `GET /api/v1/membership/my` - 我的会员状态
- `POST /api/v1/membership/orders` - 创建订单
- `GET /api/v1/membership/orders/{order_no}` - 订单详情

### 管理后台
- `GET /api/v1/admin/stats` - 系统统计
- `GET /api/v1/admin/users` - 用户列表
- `POST /api/v1/admin/messages` - 发送系统通知
- `GET /api/v1/admin/activities` - 活动日志

## 6. 应用场景

### 场景一：直播课前准备
教师登录后，在"课前准备"页面创建任务组，选择题型并添加题目。可以通过 AI 生成快速创建大量题目，或导入现有教材内容。完成任务组后保存，等待课堂使用。

### 场景二：实时互动课堂
教师进入"上课"页面，创建课堂会话并选择任务组。学生通过加入班级自动进入课堂。教师逐题发布，学生实时答题，教师可查看答题统计和排行榜，增加课堂趣味性。

### 场景三：课后作业布置
教师在"学习包"页面创建课后作业，添加词汇、句子、听力、阅读、口语等模块。设置截止时间后发布，学生在"学习"页面查看并完成，提交后教师可查看完成情况。

### 场景四：学生自主学习
学生在"免费练习"区域可以进行 A1-A2 级别的基础英语练习，无需注册即可使用。注册后可加入班级，获取教师布置的个性化学习内容。

### 场景五：竞技学习模式
教师在课堂上开启"单挑"或"班级挑战"模式，学生通过抢答获得积分，系统实时更新排行榜，激发学生学习积极性。
