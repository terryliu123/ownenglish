**报告日期：** 2026-03-27

## 项目概述

OwnEnglish 是一个直播英语老师的教学辅助平台，包含两个主要组件：
- **Client** (`client/`): React + TypeScript 前端，运行在 5173 端口
- **Server** (`server/`): FastAPI Python 后端，运行在 8000 端口
